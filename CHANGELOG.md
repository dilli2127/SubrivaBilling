## Update Types

- **Patch** (2.0.0 → 2.0.1): Bug fixes and small improvements
- **Minor** (2.0.0 → 2.1.0): New features, no breaking changes  
- **Major** (2.0.0 → 3.0.0): Breaking changes, major new features

## How Updates Work

1. **Automatic**: App checks for updates on startup
2. **Manual**: Users can check via Help → "Check for Updates"
3. **Background**: Updates download automatically
4. **Installation**: Users get prompted to restart and apply updates
